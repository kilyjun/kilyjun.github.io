Research

My current research focuses on using Physics-Informed Neural Networks (PINNs) to explore parametric solutions in composite manufacturing processes. PINNs integrate physical laws directly into the neural network architecture by incorporating governing equations as part of the loss function during training. Leveraging the universal function approximation capabilities of neural networks, PINNs enable the solution of complex partial differential equations (PDEs) with minimal data.

Previously, I took part in developing an R Shiny application for automated network meta-analysis at P&G [pdf]. I have also developed a Python software to predict shape changes in composite tooling by generating ABAQUS input files for finite element analysis [pdf].
